function basic

  • Two types: built-in adn user-defined
  • f declaration: function fname (param = 'default'){ return `hello ${param}`;} console.log(fname('arg'))
  • f expression basic: let fname = function(){return "hello"} console.log(fname())
  • f expression arrow: let fname = () => "hello"; (no parameter); let fname = name => `hello ${name}`; (one parameter); console.log(fname('ahmad')); let fname = (name, age) => {return "hello $name, $aga"}; console.log(fname('ahmad', 20)); (more than one parameter); Arrow function don't have this concept
  • arguments: function plus(){ let result = 0; for (let i = 0; i < arguments.length; i++){result +=arguments[i]; } return result; }; console.log(plus(1,2,3));
  • refactoring: making function more effective if there are variables that are not needed then delete it
  • scope: variable outside function can be accessed inside function but variable inside function cannot be accessed outside function. it's because javascript is block function
  • recursive: function factorial(n){ if(n === 0){return 1} else {return n * factorial(n-1)}}; console.log(factorail(3));; this is recursive: function inside another function, example for factorial:
  • rest parameter: function sum(...val) { let res = 0; for (v of val) res += v; return res; } console.log(sum(1, 2, 3))
  • without self-invoking: let result = (name) => `i'm ${name}`; console.log(result('ahmad'))
  • with self-invoking: console.log(((name) => `i'm ${name}`)('ahmad'));

js object

  • All Javascript value except primitives are objects. A primitive value has no properties or methods. A primitive data type is data that has a primitive value. the 7 primitive data types are: string, number, boolean, null, undefined, symbol, bigint; variables that contain many values are objects
  • Object to array: let obj = {nama: 'ahmad', umur: 20}; let arr = Object.values(obj); console.log(arr)
  • Object to string/JSON; let obj = {nama: 'ahmad', umur: 20}; let str = JSON.stringify(obj);
  • property is variable inside object; method is function inside object
  • ways to create object: object literal (basic); function declaration (object literal inside function); constructor function (using keyword this)
  • object literalA: let o = {}; o.name = 'john'; o.call = function(name){return `hello ${name} i'm ${this.name}`}; console.log(o.call('siti'));
  • object literalB: let o = {name: 'john', call: function(name){return `hello ${name} i'm ${this.name}`}}; console.log(o.call('siti'));
  • function declaration: function person(name){let o = {}; o.name=name; o.call = function(name){return `hello ${name} i'm ${this.name}`} return o}; let ahmad = person('ahmad'); console.log(ahmad.call('siti'));
